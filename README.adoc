== Print Shopping Cart by Id

[source,http]
----
GET http://localhost:8080/rest/v10/shopping-carts/{shoppingCartId}/print
----

.Example Request
[source,bash]
----
curl -X GET http://localhost:8080/rest/v10/shopping-carts/{shoppingCartId}/print
----
